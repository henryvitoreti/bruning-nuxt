# Aplicação utiliza NUXT

Leia a documentação [Nuxt documentação](https://nuxt.com/docs/getting-started/introduction).

## Rodar aplicação

- Docker

```bash
# docker-compose
docker-compose up -d
```
Acessar por: `http://localhost:8000`:

- Local

```bash
# npm
npm install

# npm
npm run dev
```
Acessar por: `http://localhost:3000`:
